# Guía para imágenes y capturas

Ubicación sugerida de imágenes: `docs/images/`

## Convenciones de nombres
- Asistencia IA: `ai_param_XX_descripcion.png` (ej. `ai_param_03_dialogo.png`).
- Antibiograma: `antibio_XX_descripcion.png` (ej. `antibio_02_aislamiento.png`).
- Usa ceros a la izquierda para mantener orden (01, 02, 03…).

## Formato y tamaño
- Formato: PNG (preferente) o SVG para diagramas exportados.
- Resolución: 1280 px de ancho mínimo (para legibilidad) o 2x si es pantalla HiDPI.
- Recorta bordes innecesarios; resalta elementos clave si es útil.

## Privacidad y datos sensibles
- No incluir datos personales reales (PHI). Usa datos ficticios en capturas.
- Revisa que no aparezcan claves o tokens en UI/console.

## Inclusión en Markdown
```md
![Título descriptivo](../images/ai_param_03_dialogo.png)
```

## Control de cambios
- Al reemplazar una imagen, conservar el nombre si el contexto sigue siendo el mismo.
- Si cambió el flujo, añade nuevas imágenes con numeración posterior y actualiza el tour.
